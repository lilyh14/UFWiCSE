Resources

Websites

HackerRank

Practice, Compete, Find Jobs

HackerRank is one of the best places to find all sorts of different coding challenges. It's also used by some of the leading tech companies as a recruiting platform!

Start Practicing

GeeksforGeeks

A computer science portal for geeks

GeeksforGeeks is incredibly useful for finding quick help on obscure algorithms you've never heard of before. It's also the #1 site to help you pass data structures.

Show it to me

r/learnprogramming

A whole subreddit for programming

An open forum dedicated to learn programming. Anyone can contribute, and the best content is voted to the top. If you're new to it, we suggest reading its wiki.

Take me there

Videos

CS50

Videos by Harvard University

Quick and easy to understand videos for a wide range of topics in computer science, such as data structures and sorting algorithms.

Start watching

mycodeschool

All of data structures in 42 videos

If you need a refresher on data structures in the form of YouTube videos, this is the one. Just be careful, these are very in-depth, so by the end you'll be a Data Structures expert.

Netflix? Please

Learn in One Video

Derek Bana's speedy lectures

If you're in a hurry and need to learn a new programming language ASAP, look no further. We suggest preparing some snacks, because some of these are long.

Binge Watch

Books

Cracking the Coding Interview

The best book for technical interviews

189 programming questions with solutions. This is perhaps the most helpful book you'll find for helping you prepare for your technical interviews.

Visit the website

Learn Python the Hard Way

Great book for learning Python

Don't let the title discourage you. You don't need prior programming experience to read this book. It takes you from zero and builds up. There's a free sample on the website.

I like free stuff

Think like a Programmer

An intro to creative problem solving

Good programming requires creativity. This book teaches you how to analyze problems, explore multiple solutions, and ultimately, think like a programmer.

Check it out
